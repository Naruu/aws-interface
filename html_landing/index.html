몇번의 클릭만으로 API를 구성하고 SDK를 배포하세요

AWSInterface 를 사용하면 Amazon Web Service 에 대한 깊은 이해 없이도 몇번의 클릭만으로 간단하게 API를 구성하고, 여러 시스템에서 동작하는 SDK를 배포할 수 있습니다.

지금 바로 시작하기
AWSI 가입하기

지금바로 시작하기를 누르고 AWS IAM 키를 제공하여 AWS Interface 를 시작합니다. 가입 후 가이드 문서를 참조하여 간단한 사용 방법과 작동 방식에 대해 알아봅니다.



더 알아보기
서비스 구성하기

대시보드에서 원하는 서비스 구성을 작성하고 저장합니다. 게시물이나 그 외 모델을 작성하고 접근 권한을 설정합니다. 또 회원가입 사용 여부와 회원가입 시 어떤 정보를 입력 받을지 그리고 비밀번호 형식과 인증 방식등을 정할 수 있습니다. 서비스 구성을 완료하면 호출가능한 API가 생성됩니다.

더 알아보기
백엔드 서비스 SDK 내려받기

앞서 구성한 백엔드 서비스의 API를 클라이언트에서 호출할 수 있도록 자동 생성된 개발 환경별 SDK를 내려받습니다. 자동 생성된 SDK는 HTTPS 프로토콜을 통해 백엔드 서비스의 API를 호출합니다.

주요기능

AWSInterface는 서비스 구성과 API 생성 그리고 SDK 배포 뿐 아니라
서비스 관리와 모니터링 기능을 포함하고 있습니다.

  • 서비스를 구성하고 업데이트 합니다.
  • 서비스 구성과 SDK를 관리합니다.
  • 서비스 로그를 모니터링 합니다.

최고의 성능을 위해

AWSInterface의 정교한 파이프라이닝과 시스템 설계가 AWS의 고가용성과 지속적 규모 조정 기능을 최대한 활용할 수 있게 도와줍니다.

완전 병렬 시스템

자동 생성된 API는 AWS Lambda에서 실행되며 함수형으로 작성됩니다. 모든 호출은 병렬로 실행되며 트래픽에 따라 규모가 자동으로 조정 되기 때문에 사용자는 트래픽을 걱정할 필요가 없습니다.

더 알아보기
간단한 인터페이스

AWS의 DynamoDB 나 Lambda 와 같은 서비스를 사용하기 위해서는 높은 수준의 배경지식이 필요합니다. 하지만 AWSInterface는 고수준으로 추상화된 인터페이스를 제공하여, AWS 서비스에 대한 별도의 이해 없이도 어플리케이션을 구성할 수 있게 도와줍니다.
